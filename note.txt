LIST TABEL
1. tabel brand (12) : samsung, xiaomi, apple, vivo, oppo, acer, realme, infinix, huawei, 
id brand(pk), nama brand

2. tabel brand list phones (5 hp per brand = 5*9 = 45)
id hape(pk), id brand(fk), nama hape, foto hape(.img)

3. tabel spek :
id spek(pk), id hape(fk), tgl rilis(tgl-bln-thn), ukuran layar(x*x inci), os(android, dll), chipset(Qualcomm dll), memory(GB RAM, GB ROM), daya baterai(Mah), kamera(MP), jaringan(2G/3G/4G), harga(RP), warna(text)

4. tabel history search
id search(pk), id hape(fk)


samsung, xiaomi, apple - meina id 1-15
